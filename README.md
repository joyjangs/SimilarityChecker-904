# 유사도 검사기

## 글자 수 검사
같은 글자수일 때, 최대 점수 60점 획득  
문자열 길이가 2배 이상 차이나면 0점  

### 부분 점수 측정 방법
- A = 긴 문자열 길이
- B = 짧은 문자열 길이

$$Gap = A - B$$
$$(1 - \frac{Gap}{B}) * 60$$

## 등장 알파벳 검사
두 문자열의 같은 종류의 알파벳이 사용되었다면 최대 점수 40점 획득  
두 문자열의 사용된 알파벳이 모두 다르다면 0점  
대문자만 취급.  

### 부분 점수 측정 방법
**𝐓𝐨𝐭𝐚𝐥𝐂𝐧𝐭** : 두 문자열에서 사용된 알파벳의 총 개수
**𝐒𝐚𝐦𝐞𝐂𝐧𝐭** : 두 문자열 중, 같은 알파벳의 개수

$$ \frac{SameCnt}{TotalCnt} * 40$$
